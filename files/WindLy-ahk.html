intersection(oWindLy)

Calculates the INTERSECTION of the given WindLy-Object and the current instance

The result only contains Windy-Objects which have been in the current instance as well as in the given WindLy-Object

Parameters

oWindyWindy-Object to be inserted

removeNonExisting

removeNonExisting()

Remove non-existing windows from the list

The result only contains Windy-Objects which do currently exist.  Non existing windows are removed from the list.

symmetricDifference

symmetricDifference(oWindLy)

Calculates the SYMMETRICDIFFERENCE of the given WindLy-Object and the current instance

The result only contains Windy-Objects which have not been in the current instance as well as not in the given WindLy-Object (Removes items which occur in both lists)

Parameters

oWindyWindy-Object to operate with

union

Calculates the UNION of the given WindLy-Object and the current instance
