_ Windy, v0.12.3

Windy provides a collection of classes, which allow a class based approach of handling windows, monitors, etc.

Following classes exist

  • Mony - Single Monitor
  • Mousy - Mouse
  • MultiMony - Multi Monitor Environments
  • Pointy - Points (geometric 2D-points) - this is mostly a helper class being used within this package, but maybe helpful anyway ...
  • Recty - Rectangles (consisting of two points) - - this is mostly a helper class being used within this package, but maybe helpful anyway ...
  • Windy - Windows
